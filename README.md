# 🎨 Hover Board — Интерактивная доска

Интерактивная доска из квадратов, которые меняют цвет при наведении мыши.
Создает красивый эффект свечения и динамики.

## 📌 Функционал

✅ Генерация сетки из квадратов
✅ Реакция на наведение мыши
✅ Случайный выбор цвета
✅ Плавные анимации
✅ Эффект "свечения"
✅ Быстрое взаимодействие

---

## 🧠 Как работает

* Создается 500 элементов (`div.square`)
* При наведении:

  * вызывается `setColor()`
  * задается случайный цвет
* При уходе мыши:

  * вызывается `removeColor()`
  * цвет возвращается обратно
* Цвет выбирается из массива:

```js
["red", "blue", "green", "yellow", "purple"]
```
